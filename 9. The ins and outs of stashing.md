# Why we need git stash

Unstaged files are not tied to any specific branch, so they will stay in your working directory and follow you when you switch branches.  
Here is a cleaner, more precise way to state this:  

- The Rule: Since Git doesn't track unstaged files yet, they remain in your local working directory. If you run git checkout , those files will simply carry over with you.  
    
- **The Exception:** Git will only block you from switching branches if an unstaged file has the _exact same name_ as a file that differs on the target branch, which would cause a merge conflict.

# Git Stash Save and Pop

Git provides an easy way to save these files so you dont have to make unnecessary and needless commits.

```bash
git stash pop
```

To remove the recently stashed changes and reapply them to your working copy. You can even apply this to reapply the changes to a different branch.
# Git Stash Apply

```bash
git stash apply
```

Changes stop in the stash and can be used across multiple branches - git pop removes it from the stash.
# Working with multiple stashes

```bash
git stash list
```

You can then reference the stash by:

```bash
git stash apply stash@{#}
```

# Dropping and clearing the stash
```
git stash drop
git stash drop stash@{#}
```
# Labs

### 1. Setup the Repository

- **Initialise** a new Git repository in a folder.
    
- **Create** a file called `diary.txt`.
    
- Inside the file, **add** the following:
    
    ```text
    I love my boss
    ```
    

### 2. Make the First Commit

- **Add and commit** the changes on the `master` branch.
    

### 3. Start the Alternate Branch

- **Create** a new branch called `the-truth` and **switch** to it.
    
- In the `diary.txt` file, **erase** the contents and replace it with:
    
    ```text
    I HATE MY BOSS
    I HATE MY BOSS
    I HATE MY BOSS
    I HATE MY BOSS
    I HATE MY BOSS
    ```
    
- **Save** the file.
    

### 4. The Panic (Stashing)

- **OH NO!** Your boss is walking towards you! Quick, **switch** over to the `master` branch!
    
- **WHATTT?** The `diary.txt` file still contains our confession? Quick, **stash** the changes before your boss sees!!
    
- Your `diary.txt` file should now only contain _"I love my boss"_.
    

### 5. Playing It Cool

- As your boss walks by, **add more lies** to the `diary.txt` file:
    
    ```text
    I love my boss
    My boss is the best leader ever
    ```
    
- **Add and commit** your changes on the `master` branch.
    

### 6. Back to Reality

- Now that your boss has left, it's safe to get back to the truth! **Switch** over to the `the-truth` branch.
    
- **Retrieve** the earlier changes that you stashed (_I HATE MY BOSS x 5_).
    
- **Add and commit** the changes on the `the-truth` branch.k
